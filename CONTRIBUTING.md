## Contributing

Please ensure all pull requests and contributions comply with the [Developer Certificate of Origin](https://developercertificate.org/).

## Adding Phrases to the Bot

To add new responses to the bot, navigate to the `phrases.ts` file. Within the array, add the phrases you'd like the bot to potentially respond with. 

## Updating the Readme

If you would like to add a reason for love to the `README.md`, please ensure that the spelling is correct. Additionally, use markdown syntax to provide a related link if applicable.
